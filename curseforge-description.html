CanYouKickNow?

Real-time interrupt cooldown tracker for Mythic+ — works even without the addon on party members.

Uses UnitClass() to build a roster automatically on dungeon entry, then confirms talent choices via Inspect. No chat spam, no addon required on others.

Key Features

  • No addon required on others — detects casts via UNIT_SPELLCAST_SUCCEEDED, no inter-addon communication needed.
  • Spec-aware via Inspect — inspects party members at dungeon start to confirm whether their interrupt is talented.
  • Auto show/hide — panel appears on Mythic+ entry, hides on exit.
  • Live cooldown bars — class-colored, updates every frame.
  • Spec correction on cast — if a Hunter uses Muzzle instead of Counter Shot, the tracker corrects itself immediately.
  • Full settings panel — /ckn config opens options for bar texture, row size, font, and more.
  • Solo test mode — /ckn test shows all 5 UI states without needing a group.
  • English / Korean UI — auto-switches by client locale.

Known Limitations

  • Warlock interrupts are pet-based and not exposed via standard player unit events — tracking is not possible with the current WoW API.
  • Death Knight Coldthirst (−3s per successful interrupt) is event-driven and cannot be tracked.
  • Talented interrupts show ? until Inspect completes (~30 yard range required) or the player casts the interrupt.

CanYouKickNow? (한국어)

M+ 파티 차단기 쿨타임 실시간 추적 애드온 — 상대방 애드온 없이도 동작합니다.

UnitClass()로 파티원 클래스를 자동 감지하고, Inspect로 특성 보유 여부를 확인합니다. 공대 채팅 스팸 없음.

주요 기능

  • 상대방 애드온 불필요 — UNIT_SPELLCAST_SUCCEEDED로 감지, 애드온 간 통신 필요 없음.
  • Inspect 기반 특성 확인 — 던전 진입 시 자동으로 파티원 검사, 차단 특성 보유 여부 확정.
  • 자동 표시/숨김 — M+ 진입 시 자동 표시, 이탈 시 자동 숨김.
  • 실시간 쿨타임 바 — 직업 색상, 매 프레임 업데이트.
  • 스펙 자동 교정 — 실제 사용 스킬 기준으로 스펙 추정 자동 수정.
  • 설정 패널 — /ckn config로 바 텍스처, 크기, 폰트 등 조정.
  • 솔로 테스트 모드 — /ckn test로 혼자서도 UI 전체 상태 확인.
